# springboot_demo_app

## 无忧商城

> 一个基于vue2的商城APP前端项目,使用 vant UI
> 参考天猫商城的购物流程：用户从注册开始，到完成登录，浏览商品，加入购物车，进行下单，确认收货，等一系列操作
